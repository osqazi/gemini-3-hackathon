# Theme Images

This directory contains royalty-free theme images for the RecipeRAG application.

## Images Included

- `hero-background.jpg` - Background image for the hero section
- `ingredients-grid.jpg` - Grid of fresh ingredients
- `cozy-kitchen.jpg` - Cozy kitchen scene
- `prepared-meal.jpg` - Beautifully plated prepared meal
- `recipe-card-bg.jpg` - Background for recipe cards
- `avatar-placeholder.jpg` - Placeholder for user avatars

## Source

All images in this directory are sourced from Unsplash and Pexels with appropriate licenses for commercial use.

## Image Specifications

- Maximum size: 100KB each
- Optimized for web use
- Appropriate for food/kitchen theme
